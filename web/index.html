Map It

Spring 2014

Map It helps you find things in the library

The Harvard Library has miles of shelves. Map It will tell you which one is home to the item you need.

Helping Users, Helping Staff

We want to improve the user experience of searching for an item in the library.

  • » Users will be better able to locate items for their research and scholarship
  • » Users will be able to navigate the stacks and collections more independently
  • » Users will spend less time locating materials and staff will spend less time directing them
