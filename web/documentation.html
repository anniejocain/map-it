Map It

Spring 2014

The API is simple

Tell Map It the library, collection code and call number. You'll get a level, shelf and link to a map in return.

Form a request

http://librarylab.law.harvard.edu/map-it/api/locate/lam/gen/PS3620.A957%20D38%202011

Get JSON response

{
    "floor": "3",
    "range": "53",
    "maplink": "http://librarylab.law.harvard.edu/map-it/map/lam/3/53"
}
